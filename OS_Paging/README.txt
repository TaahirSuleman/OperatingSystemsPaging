
## Usage
To run the program please enter the following command:
python3 paging.py [number of frames]
Where number of frames is the size parameter that specifies the number of frames available in physical memory.
You will then be prompted to enter a size for the random page-reference string - please enter an integer here that represents your desired reference string size.
The random page-reference string will be pasted for your use in determining the correctness of the algorithms.
